## react-with-ng2

simple example how to run ReactJS component inside angular2 Component

**Stack :**
* [react](https://facebook.github.io/react/)
* [redux](http://redux.js.org/)
* [webpack](https://webpack.github.io/)
* [ng2](https://angular.io/)

## Install

```
npm i
```

## Run
 
```
npm start
```
